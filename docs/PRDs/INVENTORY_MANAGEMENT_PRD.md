# PRD: Inventory Management

## 1. Overview
Tracks the physical stock of Scooters, Batteries, Chargers, and Spares across multiple showrooms.

## 2. Key Features
- **Categorization**: Group items by type (Vehicle, Battery, Spare Part).
- **Showroom Partitioning**: Track stock levels per location (Main, North, South, West).
- **Auto-Deduction**: Integrated with Sales and Service modules to automatically update stock quantities.
- **Low Stock Alerts**: Visual "Red Labels" when an item's quantity drops below its predefined `reorder_level`.
- **Bulk Upload**: Admin feature to import thousands of spare parts via CSV.

## 3. Inventory KPIs
- Total Valuation (Stock value).
- Turnaround time for spare parts.
- Out-of-stock frequency.
